Ticker BTC-e
===========
Ticker for BTC/USD (from BTC-e).
(can change pairs in sources)
Windows only. Need .NET 4.5.

Features:
- All time window stay on top with transparent
- Candlestick chart (candle = 1 min, last 60 minutes)
